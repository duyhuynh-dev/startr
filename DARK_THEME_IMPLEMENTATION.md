# Dark Theme Implementation Plan
## Charcoal & Gold - Fancy & Professional

### Color Palette

**Primary Colors:**
- Background: `#0F172A` (Slate-900) - Deep dark
- Surface: `#1E293B` (Slate-800) - Cards/panels
- Elevated: `#334155` (Slate-700) - Hover states

**Accent Colors:**
- Primary: `#F59E0B` (Amber-500) - Gold accent
- Hover: `#D97706` (Amber-600) - Darker gold
- Light: `#FCD34D` (Amber-300) - Light gold highlights

**Text Colors:**
- Primary: `#F1F5F9` (Slate-100) - Light text
- Secondary: `#CBD5E1` (Slate-300) - Muted text
- Accent: `#F59E0B` (Amber-500) - Gold text

**Borders:**
- Default: `#475569` (Slate-600) - Subtle borders
- Accent: `#F59E0B` (Amber-500) - Gold borders

### Implementation Strategy

1. Update Button component with gold accents
2. Update Navbar with dark background and gold highlights
3. Update Card components with dark surfaces
4. Update page backgrounds to dark
5. Update text colors throughout
6. Add subtle gold accents for premium feel
